To set this device up as the display ensure that it is connected to the same WIFI network as the display device.

Note that the WIFI network does not need to be connected to the public internet. This can be a standard router, or even setting one of the devices to act as a mobile hotspot.

Once the device is connected to the WIFI start the connection on the display device. Once the QR Code is visible tap the button below and scan the code to complete the connection.
