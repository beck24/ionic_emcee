Step 1: Connect to WIFI

Connect this device to a WIFI network. The network does not need internet access. It can be a simple wireless router plugged into a wall, or a mobile hotspot provided by this or another device. All that matters is that this device and the remote device are connected to the same network.

Swipe left for next instruction.

Step 2: Connect to Remote

Tap on the "Connect" button. This will make the device available for connection. A QR code (pictured below) will be visible when the device is ready. Scan this code with the remote device to complete the connection.


Example QR Code

Step 3: Use Display

On successful connection to the remote device the display will turn into a countdown timer. The remote device can be used to control the timer in terms of timer duration, font size, timer start/stop/reset, and temporary popup messages.

In this mode the display device has limited user interactivity, it acts a simple display.

Swipe left for next instruction.

Step 4: Shut Down

When the device has finished being used you can disconnect it from the remote device and restore the app to its initial state.

Tap on the dislay 5 times in rapid succession. A screen will pop up with a button to reset the application. Tapping this button will disconnect the device from the remote and reset the app.
